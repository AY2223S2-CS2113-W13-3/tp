Command 'bye': This command is used to exit the program.

Command 'add': This command adds a food product to the food supply tracker.
Format: add -n PRODUCT_NAME -e EXPIRY_DATE

Command 'remove': This command removes the food product from the food supply tracker based on its index.
Format: remove INDEX

Refer to our user guide for more in-depth details on how to use our system:
